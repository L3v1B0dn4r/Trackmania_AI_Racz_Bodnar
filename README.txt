In this document the description of the project can be found with a basic idea and a step-by-step guide on how to run the AI model.

The following AI was developed with the purpose if successfully pass the "Artificial_Intelligence_and_autonomous_vehicles" subject
The general idea is an AI that can learn and successfully drive on any given track in Trackmania Nations Forever.

Step-by-Step guide:

1. Download the necessary dependencies.
2. Open and run "controls.py" to define the controls for the AI.
3. Open and run "screen_capture.py" to see the "AI window".
4. Open Trackmania Nations Forever in windowed mode and position it inside the "AI window".
5. Open and Run "trackmania_env.py" to establish the environment.
6. Open and Run "train_rl.py" and switch to the game to start training.
7. Open and Run "ai_driver.py" after training to start the AI.
